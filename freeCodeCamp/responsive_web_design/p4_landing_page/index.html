For easy navigation on the World Wide Web

Features

  • It offers information 24 hours a day, and it can be accessed as long as there is a connection.
  • It is public by nature.
  • It doesn't belong to any entity, and it's universal and decentralized, so there are no control points at the hierarchical level.

How It Works

  • A device connects to the internet through a modem or router.
  • When a user types a web address into a browser, the device sends a request to a DNS server to find the IP address for the website.
  • The DNS server looks up the IP address and sends it back to the device.

Pricing

  • Fiber: $21–$300, with equipment fees of $50–$100
  • Cable: $20–$120, with equipment fees of $10–$15
  • DSL: $50–$65, with equipment fees of $10
